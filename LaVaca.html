Tipos de Malware Descripcion Como disuadirlo
RANSOMWARE Cifra archivos y exige rescate para liberarlos. Crear copias de seguridad recurrentes.
WORMS Se replican y propagan automáticamente por redes explotando vulnerabilidades. Uso de firewalls para bloquear tráfico, configurar segmentación de red.
SIDE CHANNEL ATTACKS Explota filtraciónes físicas del hardware como tiempo de ejecución, señales electromagneticas para robar datos sensibles. Uso de hardware y software con protecciones contra atques fisicos, deshabilitar sensores o interfaces innecesarias.
