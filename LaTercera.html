1. Malware Biológico

Una forma experimental donde malware fue codificado en cadenas de ADN, demostrando que es posible almacenar código malicioso dentro de organismos biológicos.

2. USB Killer

Dispositivo USB que libera una descarga eléctrica para dañar físicamente los componentes internos de una computadora. No necesita software para causar daño.

3. Stuxnet

Un malware diseñado para sabotear instalaciones nucleares iraníes. Fue uno de los primeros ejemplos de ciberarma con impacto físico en el mundo real.

Curiosidades

  • Algunos malware se activan por sonidos específicos o frecuencias.
  • Existen virus diseñados solo para mostrar arte ASCII en pantallas.
  • Otros se esconden en chips de hardware y firmware.
